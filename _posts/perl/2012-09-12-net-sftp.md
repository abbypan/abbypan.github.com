### Net::SFTP 提示 Can't bind socket to port 1023: Address already in use

Can't bind socket to port 1023: Address already in use

初始化的时候加上``privileged=>0``参数即可

{% highlight perl %}
my $sftp = Net::SFTP->new(
       $ip,
       user => $user,
       password => $password,
       ssh_args => [ port => $port, privileged=>0 ],
 );
{% endhighlight %}
